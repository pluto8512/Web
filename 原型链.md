###原型链
凡是对象都有原型

原型对象也有原型

一旦定义了构造函数就有空两个部分
```javascript
var p = new Person();
// p默认连接到Preson.prototype上
// p-->Person.prototype
// 或
// p-->_proto_
// 控制台测试
p: Person
Person.prototype: Object
->constructor: function Person()
p._proto_ === Person.prototype: true
Person.prototype._proto_: Object
->constructor: function Object()
person.protype._proto_._proto_:null
```
