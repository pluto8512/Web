#作用域链
######作用域链规则
-> 声明是作用域链
      链指的是访问规则:
      function foo() {
         console.logg( num ); 
      }
      ---------------
      function func() {
         function foo() {
            console.log( num );
         }
         foo();
      }
      一层一层的去找
> 由于这种一环套一环的访问规则,这样的作用域构成一个链式结构,称其为作用域链
