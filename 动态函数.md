###动态函数 Function

接下来定义和调用动态函数

Function是一个构造函数

```javascript
var one = new Function("参数1","参数2",..."参数N",fn);
```

案例:

```javascript
//求从第一个数字到第二个数字的和
function fn1 (min,max) {
   var sum = 0;
   for(var i= min; i <=max; i++) {
      sum += i;
   }
   return sum;
}
//将fn1改写成new Function()的形式
var fn2 = new Function("min","max","
  var sum = 0 ;
  for (var i = min ; i <= max ; i++){
    sum += i ;
  }
  return sum;
");
```
